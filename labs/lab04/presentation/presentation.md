# Цель работы

## Основная цель

Установить и настроить среду моделирования сетей **GNS3**,  
а также импортировать и проверить работоспособность  
маршрутизаторов **FRR** и **VyOS**.

# Среда моделирования GNS3

## Используемое ПО

- GNS3-all-in-one  
- GNS3 VM  
- VirtualBox / VMware  
- Маршрутизаторы:
  - FRR (Free Range Routing)
  - VyOS

# Установка GNS3

## Проверка запуска GNS3 VM

- Проверена корректность запуска виртуальной машины  
- Определены версии:
  - GNS3 server
  - Ubuntu
  - QEMU  
- Назначен IP-адрес и порт контроллера
