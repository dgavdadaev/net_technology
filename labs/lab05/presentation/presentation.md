# Цель работы

## Основная цель

Построение и исследование простейших сетевых топологий  
в среде моделирования **GNS3** с использованием:

- Ethernet-коммутаторов  
- оконечных устройств VPCS  
- маршрутизаторов **FRR** и **VyOS**  

А также анализ сетевого трафика с помощью **Wireshark**.

# Простая сеть на базе коммутатора

## Топология сети

- Два оконечных устройства VPCS  
- Один Ethernet-коммутатор  
- Все узлы находятся в одной подсети  

![Топология сети](Screenshot_1.png){ width=70% }

## Анализ ARP

- Зафиксированы gratuitous ARP-запросы  
- Используются для:
  - проверки уникальности IP-адреса  
  - объявления соответствия IP–MAC  
- Передача осуществляется на широковещательный MAC-адрес  

![ARP-трафик](Screenshot_5.png){ width=70% }

## Анализ ICMP

- Выполнены ICMP Echo Request / Reply  
- Подтверждена корректная доставка пакетов  
- TTL и идентификаторы соответствуют стандарту  

![ICMP-трафик](Screenshot_7.png){ width=70% }

## Анализ UDP

- UDP:
  - отсутствует установление соединения  
  - используется порт 7 (Echo)  

![UDP](Screenshot_8.png){ width=70% }

## Анализ TCP

- TCP:
  - выполнено трёхстороннее рукопожатие  
  - передача данных и корректное завершение сессии  

![TCP](Screenshot_9.png){ width=70% }
